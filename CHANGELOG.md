# CHANGELOG

## 친구찾기 챌린지 V6 - 2026-07-01

### 수정
- `친구찾기 시작` 버튼을 눌러도 반응이 없던 문제를 수정했습니다.
- 미션 출현기록/중복 방지 deck 생성 중 참조 오류가 발생하지 않도록 `shuffle` 함수를 전역화했습니다.
- 시작 중 예외가 발생하면 콘솔과 안내창으로 확인할 수 있게 처리했습니다.
- 실행 가능한 미션이 0개일 때 조용히 멈추지 않고 안내문을 표시합니다.

### UI 개선
- 설정 화면을 `게임 설정`과 `테스트 조작` 영역으로 정리했습니다.
- `미션 횟수`, `제한시간`, `미션 범위`를 세로 배치해 수업 시작 화면을 단순화했습니다.
- `리모컨 새창으로 열기` 버튼이 애매하게 2줄로 깨지지 않도록 `리모컨 열기`로 줄이고 줄바꿈을 방지했습니다.

### 유지
- 쉼표(,)로 여러 성공자 입력 시 각각 +1점 처리
- 효과음 크기 단계: 끄기 / 보통 / 크게 / 매우 크게
- 최근 4시간 미션 출현기록 우선 제외 및 후보 부족 시 재출현 허용
- 루트 `version.json` 및 `public/version.json` 포함
