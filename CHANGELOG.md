# CHANGELOG

## 친구찾기 챌린지 V4 - 2026-07-01

### 수정
- 시작 화면에서 미션 관리 목록 제거
- 미션 추가/수정/즐겨찾기/X 제외/숨김/복구를 별도 미션 관리자 화면으로 분리
- 기본 미션 버튼명을 `삭제`에서 `숨김`으로 변경
- 기본 미션은 실제 삭제하지 않고 숨김 처리
- 사용자 추가 미션만 삭제 가능하도록 구분
- 미션 문구를 전체화면에 더 꽉 차게 표시
- 한글 단어가 중간에 끊기지 않도록 의미 단위 줄바꿈 적용
- 보조 이미지/이모지가 미션 텍스트를 가리지 않도록 레이아웃 분리
- 미션 키워드에 맞는 보조 아이콘을 우선 표시하고, 어려운 경우 기본 친구찾기 아이콘으로 대체
- version.json의 version 값을 `4`로 단순 표기

### 유지
- `/version.json` 배포 구조 유지
- 리모컨 새창 열기
- QR 리모컨 연결
- 효과음 중심 카운트다운/미션 진행/종료 구조
- 친구찾기 약속 화면 수동 진행

### 운영 메모
- 정식 고객용 쌤플레이에는 반영하지 않음
- 게임실험실 독립 테스트판에서 검증 후 안정화 시 정식판 반영 검토
