# CHANGELOG

## 친구찾기 챌린지 V5 - 2026-07-01

- 성공자 기록 쉼표 분리 처리 추가
  - 예: `순신, 임당` 입력 시 순신 +1, 임당 +1
  - 앞뒤 공백 제거 및 같은 입력 내 중복 이름 1회 처리
- 성공자 입력 안내문구 추가
  - `여러 명은 쉼표(,)로 구분해 입력하세요. 예: 순신, 임당`
- 결과 화면 개선
  - 사람별 누적 점수표 표시
  - 점수순 정렬
  - 1등 트로피 배지 표시
  - 타이틀/이름/점수 영역 분리
- 효과음 개선
  - 기본값을 학원 환경 기준 `크게`로 변경
  - `끄기 / 보통 / 크게 / 매우 크게` 단계 추가
  - 종료음은 다른 효과음보다 더 강하게 재생
- 미션 중복 출현 방지
  - 최근 4시간 내 출현 미션 우선 제외
  - 후보 부족 시 재출현 허용
  - 바로 연속 재출현은 최대한 회피
  - 출현기록 초기화 버튼 추가
- TV 미션 화면 개선
  - 좌우 빈 공간을 보조 이미지 안전영역으로 활용
  - 텍스트와 보조 이미지 겹침 방지 유지
