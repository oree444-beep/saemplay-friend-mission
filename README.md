# 친구찾기 챌린지 V1

쌤플레이 게임실험실 하위 항목에 등록할 배포 전 테스트용 독립 게임입니다.
정식 고객용 쌤플레이의 기존 리모컨, 점수, 타이머, 문제은행 기능은 건드리지 않습니다.

## 포함 기능

- 친구찾기 챌린지 테스트판
- 친구찾기 약속 화면
- 미션 횟수/제한시간/미션 범위 설정
- 기본 미션 60개
- 즐겨찾기/제외/추가/수정/삭제
- 카운트다운/제한시간/다음 미션 준비
- 성공자 기록/점수표

## TeamHR 등록 위치

고객용 쌤플레이 테스트버전 > 쌤플레이 게임실험실 > 친구찾기 챌린지
