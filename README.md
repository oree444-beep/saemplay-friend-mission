# 쌤플레이 친구찾기 챌린지 V5

쌤플레이 게임실험실 안에서 배포 전 테스트하는 친구찾기 챌린지 독립 테스트판입니다.
정식 고객용 쌤플레이의 기존 리모컨, 점수판, 타이머, 문제은행 구조를 건드리지 않습니다.

## V5 핵심 수정

- 성공자 기록에서 `순신, 임당`처럼 쉼표로 입력하면 각각 1점씩 누적합니다.
- 입력창에 `여러 명은 쉼표(,)로 구분` 안내문구를 추가했습니다.
- 결과표를 사람별 누적 점수 기준으로 정렬하고 1등에는 트로피 배지를 표시합니다.
- 효과음 크기를 `끄기 / 보통 / 크게 / 매우 크게`로 선택할 수 있습니다.
- 기본 효과음은 학원 환경에서 잘 들리도록 `크게`로 설정했습니다.
- 최근 4시간 내 출현한 미션은 우선 제외해 연속 게임 중 반복 출현을 줄였습니다.
- 후보 미션이 부족하면 재출현은 허용하되, 바로 연속 출현은 최대한 피합니다.
- 미션 화면 좌우 빈 공간을 보조 이미지 안전영역으로 활용했습니다.

## 배포 확인

1. GitHub 저장소 최상단에 이 폴더 안의 파일들을 업로드합니다.
2. Vercel 배포가 완료되면 실행 주소를 확인합니다.
3. `/version.json`에서 `친구찾기 챌린지 V5`와 `"version": "5"`가 보이는지 확인합니다.
4. TeamHR에서 버전 자동 확인을 실행합니다.
