# 친구찾기 챌린지 V6

쌤플레이 게임실험실 안에서 배포 전 테스트하는 친구찾기 챌린지 독립 테스트판입니다.
정식 고객용 쌤플레이의 리모컨, 점수판, 타이머, 문제은행 구조를 건드리지 않고 별도 테스트판으로 운영합니다.

## V6 핵심 수정

- `친구찾기 시작` 버튼 무반응 문제 수정
- 미션 출현기록/중복 방지 로직에서 시작 함수가 멈추지 않도록 랜덤 섞기 함수를 전역 처리
- 시작 시 실행 가능한 미션이 없으면 안내창 표시
- 설정 화면을 `게임 설정`과 `테스트 조작` 2개 영역으로 재배치
- `미션 횟수`, `제한시간`, `미션 범위`를 세로형 한 카드로 정리
- `리모컨 새창으로 열기` 버튼은 `리모컨 열기`로 줄여 한 줄 표시
- V5의 쉼표 성공자 집계, 효과음 단계, 미션 재출현 방지, 결과표 개선 유지

## 배포 확인

1. Vercel 실행 주소에서 화면 타이틀이 `친구찾기 챌린지 V6`인지 확인합니다.
2. `/version.json` 주소가 열리는지 확인합니다.
3. TeamHR에서 버전 자동 확인을 눌러 V6이 읽히는지 확인합니다.
