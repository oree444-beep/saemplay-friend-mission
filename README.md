# 친구찾기 챌린지 V7

쌤플레이 게임실험실에서 배포 전 테스트하는 친구찾기 챌린지 독립 테스트판입니다.

## V7 핵심 수정
- 미션 문구가 화면 밖으로 잘리는 치명 표시 버그를 수정했습니다.
- 미션 문구를 공백/의미 단위로 자동 줄바꿈하고, 길이에 따라 폰트 크기를 자동 조정합니다.
- `오늘 웃은 친구와 인사하기` 같은 문구가 일부만 보이지 않고 전체 문장이 표시되도록 개선했습니다.
- 좌우 보조 이미지는 동일한 밝기와 상태로 표시되도록 수정했습니다.
- `양말 색이 비슷한 친구 찾기`는 옷 이미지가 아니라 양말 이미지가 우선 나오도록 구체 명사 우선 매칭을 적용했습니다.
- V6의 시작 버튼 수정, 효과음 단계, 쉼표 성공자 집계, 미션 중복 방지 기능은 유지합니다.

## 배포 확인
1. 실행 주소에서 상단 버전이 V7인지 확인합니다.
2. `/version.json` 주소가 열리고 `version: "7"`인지 확인합니다.
3. TeamHR에서 버전 자동 확인을 눌러 V7을 읽는지 확인합니다.
