# 쌤플레이 친구찾기 챌린지 V2

쌤플레이 정식 고객용에 바로 반영하지 않고 `고객용 쌤플레이 테스트버전 > 쌤플레이 게임실험실 > 친구찾기 챌린지`에서 검증하는 독립 테스트판입니다.

## V2 핵심 수정

- 화면에 `쿵!`, `띵...띵...띵...` 텍스트를 표시하지 않고 실제 효과음으로 처리
- 미션 시작/카운트다운/진행 중 타이머/종료 효과음 구조 추가
- 브라우저 자동재생 제한 대응을 위해 버튼 클릭 시 오디오 unlock 처리
- `리모컨 새창으로 열기` 버튼 추가
- QR 리모컨 연결 기능 유지
- 미션 문구를 OX퀴즈/연상퀴즈처럼 더 크게 표시
- 작은 움직이는 이모지 캐릭터 추가
- 기본 미션 삭제는 숨김 처리, 기본 미션 복구 버튼 제공

## 안전 방향

아이들이 뛰지 않고 걸어서 조건에 맞는 친구를 찾는 게임입니다.
시작 전 약속 화면은 자동으로 넘어가지 않고 선생님이 `다음 진행`을 눌러야 합니다.

## 실행

```bash
npm install
npm run dev
```

## 배포

Vercel에 업로드 후 `/version.json`이 정상 표시되는지 확인합니다.
