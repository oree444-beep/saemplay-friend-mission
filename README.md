# 쌤플레이 친구찾기 챌린지 V4

쌤플레이 정식 고객용에 바로 반영하지 않고 `고객용 쌤플레이 테스트버전 > 쌤플레이 게임실험실 > 친구찾기 챌린지`에서 검증하는 독립 테스트판입니다.

## V4 핵심 수정

- 게임 시작 화면에서 `미션 관리` 목록 제거
- `미션 관리자`를 별도 모달 화면으로 분리
- 기본 미션은 `삭제`가 아니라 `숨김` 처리
- 숨긴 기본 미션 보기/복구 기능 추가
- 사용자 추가 미션만 실제 삭제 가능
- 미션 문구를 TV 화면에 더 크게, 꽉 차는 느낌으로 표시
- `키가 비슷한 친 / 구끼리`처럼 한글 단어가 중간에 끊기지 않도록 의미 단위 줄바꿈 적용
- 보조 이모지/이미지는 미션 이해를 돕는 용도로 사용하되 미션 텍스트와 겹치지 않도록 별도 영역에 배치
- `version.json`의 `version` 값은 `4`처럼 단순 숫자 표기

## 안전 방향

아이들이 뛰지 않고 걸어서 조건에 맞는 친구를 찾는 게임입니다. 시작 전 약속 화면은 자동으로 넘어가지 않고 선생님이 `다음 진행`을 눌러야 합니다.

## 배포 후 확인 순서

1. Vercel 실행 주소 확인
   - `https://saemplay-friend-mission.vercel.app`
2. version.json 직접 확인
   - `https://saemplay-friend-mission.vercel.app/version.json`
3. version.json에서 `친구찾기 챌린지 V4` 또는 `"version": "4"` 표시 확인
4. TeamHR에서 `버전 자동 확인` 클릭

## 운영 메모

- 정식 고객용 쌤플레이에는 반영하지 않습니다.
- 게임실험실 독립 테스트판에서 충분히 검증한 뒤 안정화 시 정식판 반영을 검토합니다.
